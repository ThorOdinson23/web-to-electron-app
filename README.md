# Web to Electron app

Transform any web to an Electron app in few steps

##Steps: 

* Clone or fork this repository

```bash
$ git clone https://github.com/aralroca/web-to-electron-app.git
````

* Install dependencies:

```bash
$ sudo npm install
```

* Example:

```bash
$ electron . 'Google' http://www.google.es
````
